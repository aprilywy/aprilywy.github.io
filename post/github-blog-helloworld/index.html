GITHUB個人網站創立

  • 2022/10/24
  • 閱讀時間 1 分鐘

這幾天終於用好了之前一直想架設的網站，太開心了（撒花

之前開始學習寫程式的時候，就一邊找筆記軟體和可以發布文章和大家討論的地方，試過一些平台都覺得不太適合我，也嘗試過WordPress，然後好多功能沒用到，還花了不少錢，因此就想上網找其他自架網站的方法，最好還是可以支援Markdown語法，讓我可以使用我寫筆記的方式寫文章，

剛好看到Hahow的課程有「Github 免費架站術！輕鬆打造個人品牌」，所以毫不猶豫的就買下去了，而且老師講課也很親切，更讓我有實際做出網站的動力，覺得買得很值得<3

老師在課堂上是使用相對好上手的HUGO模板，手把手帶我們修改，把網站修改成具個人特色，沒什麼基礎也可以做得很好看，如果原本就有基礎的話可以修改的更美觀更精緻，這個網站應該還可以吧，嘿嘿:D

目前還不太會寫文章，所以表達上顯得亂七八糟的，希望未來我文章的表達能力和寫程式的能力都可以進步，我會好好加油的！！
